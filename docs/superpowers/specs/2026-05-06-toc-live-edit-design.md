# 文档目录大纲 + 实时编辑模式 设计文档

> 日期: 2026-05-06
> 状态: 已确认

## 概述

为笔记和计划编辑器新增两个功能：
1. **文档目录大纲** — 解析 Markdown 标题层级，提供可点击跳转的目录导航面板
2. **实时编辑模式** — Typora 风格的行内 Markdown 渲染编辑体验

同时将现有编辑/预览两段式切换扩展为三段式。

---

## 一、编辑模式切换

### 模式定义

从左到右三段式按钮：

| 按钮 | 模式 | 编辑 | 渲染 | 适用场景 |
|------|------|------|------|----------|
| ⚡ | 实时模式（默认） | 可编辑 | 光标所在块显示源码，其余块渲染 | 日常写作 |
| ✏️ | 代码模式 | 可编辑 | 不渲染，纯 Markdown 源码 | 批量编辑、粘贴内容 |
| 👁 | 预览模式 | 只读 | 全文渲染 | 阅读和检查排版效果 |

### 实时模式交互细节

- 点击已渲染的段落 → 该块切换为 Markdown 源码编辑状态
- 光标离开该块 → 自动渲染为最终效果
- 支持行内渲染的元素：标题、粗体、斜体、链接、行内代码、图片
- 作为整体块切换的元素：列表（整体）、代码块（整体）、引用块（整体）
- 表格、分隔线等也作为整体块

### UI 变更

- 现有工具栏中 `Pencil` / `Eye` 两段式切换扩展为三段式
- 按钮顺序：⚡ 实时 | ✏️ 代码 | 👁 预览
- mode 类型从 `'edit' | 'preview'` 扩展为 `'live' | 'edit' | 'preview'`
- 默认模式改为 `'live'`

### 影响组件

- `NoteEditor` — 添加 live 模式状态和切换
- `MarkdownEditor` — 保留用于代码模式
- 新增 `LiveMarkdownEditor` — 实时模式编辑器组件
- `MarkdownPreview` — 保留用于预览模式

## 二、文档目录大纲

### 布局

三栏结构，从左到右：

```
[ 独立侧栏 (TOC) ] | [ 编辑区 (宽度不变) ] | [ 图标栏 ]
```

- 目录面板从最左侧推入，编辑区宽度完全不受影响
- 独立侧栏是可扩展区域，未来可复用于其他内容（信息卡片等）
- 面板宽度约 120px

### 触发/关闭

| 方式 | 说明 |
|------|------|
| ☰ 工具栏按钮 | 点击切换目录面板显示/隐藏，激活时按钮高亮（蓝色背景 + 蓝色边框） |
| ✕ 关闭按钮 | 目录面板顶部右上角的关闭按钮 |
| Ctrl+Shift+O | 快捷键辅助切换 |
| 自动关闭 | 从编辑器返回列表时自动关闭目录 |

### 目录内容

- 解析当前文档的 H1-H6 标题
- **默认展示 H1-H3 三级**，在设置中可配置显示层级（1-6）
- 标题层级通过左侧缩进体现（每级 +8px）
- 当前光标所在标题高亮（蓝色文字 + 浅蓝背景）
- 标题过长时省略号截断
- 面板内可滚动
- 无标题时显示"暂无标题"空状态
- 文档内容变化时实时更新目录

### 交互

- 点击标题 → 编辑器滚动到对应位置
- ☰ 按钮位于工具栏中，模式切换按钮的左侧
- 编辑器和预览中均可打开目录

### 影响组件

- `NoteEditor` — 添加目录面板和触发按钮
- `PlanEditor`（如有内容编辑）— 同上
- 新增 `TableOfContents` — 目录面板组件
- `Sidebar` 布局 — 在编辑器左侧预留独立侧栏位置
- 设置页 — 添加目录显示层级配置项

## 三、设置项

在设置页添加：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| tocMaxLevel | number | 3 | 目录显示的最大标题层级 (1-6) |

## 四、技术要点

### 实时模式实现思路

- 将 Markdown 文档按块（block）拆分：标题、段落、列表、代码块等
- 每个块独立管理渲染/编辑状态
- 光标所在块显示为 textarea（源码），其余块显示为渲染后的 HTML
- 块切换时保持光标位置和滚动位置

### 目录解析

- 从 Markdown 源码中提取 `#` 标题行
- 支持实时解析（内容变化时重新提取）
- 在实时模式和代码模式中均可解析（从源码提取）
- 在预览模式中从渲染后的 DOM 提取或从源码提取

### 独立侧栏布局

- 在 Sidebar 组件中，编辑器面板左侧增加可选的侧栏插槽
- 该插槽通过状态控制显示/隐藏
- 插槽内容可替换（当前为 TOC，未来可扩展）
